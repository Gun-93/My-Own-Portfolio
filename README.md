#My Portfolio Website

Welcome to my personal portfolio!  
This project showcases my skills, projects, and experience as a web developer in a clean and responsive design.

---

## Features

- **Responsive Design** – Works on desktop, tablet, and mobile screens.
- **Interactive Navigation** – Plays sound effects when switching sections.
- **Smooth Animations** – Modern, smooth UI transitions.
- **Contact Form** – Allows visitors to reach out to me easily.
- **Projects Showcase** – Displays my featured projects with descriptions.

---

## Tech Stack

- **HTML** – Structure
- **CSS** – Styling (Flexbox + Media Queries for responsiveness)
- **JavaScript** – Interactivity
- **Sounds** – Custom sound effects for navigation
